# Visa CRM Production Checklist

- Meta webhook URL configured in Meta App.
- Meta App Secret configured.
- Page Access Token configured.
- Scheduler enabled.
- Lead Intake Queue records move to `Processed`.
- Customer360 matching links existing Customer or CRM Lead.
- Counselor assignment creates logs.
- Communication Event is created for each intake.
- Follow-up ToDo is created.
- Production Health page loads.
- Meta Diagnostics page loads.
- Scheduler Diagnostics page loads.
- Portal pages `/visa-portal` and `/document-upload` load.
- Print Formats exist without tutorial CSS or sample text.
